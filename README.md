# RP Django CLI

Rapid Django project generator - Create Django projects from templates in seconds.

## Installation

```bash
pip install rp-django-cli
```

## Usage

```bash
# Create a new Django project
rp create myproject

# Create project in a specific folder
rp create myproject --folder custom-folder

# Use a specific template
rp create myproject --template default
```
